Tech Stack
This portfolio is built with modern web technologies to ensure speed and responsiveness:

Framework: Vite

Styling: Tailwind CSS v4

Interactions: JavaScript & jQuery

Animations: Animate.css

Deployment: Vercel

Key Features
Fully Responsive: Optimized for mobile, tablet, and desktop views.

Dark Mode Aesthetic: High-contrast "Deep Blue" theme for better readability.

Modern UI: Glassmorphism effects and smooth scrolling.

Project Showcases: Highlights my work on BreathCoder AI, Smart Bus System, and Guardian Paws.
